# nodejs-mysql-restapi

API - CRUD, employees

## Built with 🛠️

- Node Js
- Express
- MySql
- Mysql2

## Installation 🔨

1. **Clone Project**: 
```
git clone https://github.com/rafeldev/nodejs-mysql-restapi
```

2. **Install dependencies**: 
```
npm install
```

3. **execute program in development mode**: 
```
npm run dev
```
